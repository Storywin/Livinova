You are a senior full-stack software architect and product engineer.

Build a production-ready web platform for a property marketplace brand named "Livinova".

PROJECT NAME
Livinova

BRAND POSITIONING
Livinova is a trusted property marketplace focused on developer-listed properties that are already integrated with Smart Living / Smart Home features and manually verified by our internal team.
Other property developers can also apply to:
1. list their properties,
2. request smart living / smart home integration for their projects,
3. advertise their projects on the platform.

The platform must look modern, premium, trustworthy, scalable, and SEO-friendly.

PRIMARY GOALS
1. Build a public-facing property listing website.
2. Build a developer portal for property developers to submit listings and request smart home integration.
3. Build an internal admin panel for manual verification, approvals, and operational management.
4. Build mortgage simulation tools for Indonesian banks.
5. Build a promotional/advertising system for featured projects and sponsored listings.
6. Build the system with clean architecture and production-ready code.

CORE BUSINESS RULES
1. Only verified smart living / smart home properties can be published as fully active verified listings.
2. Developers can submit properties before verification, but they remain in pending review until approved by admin.
3. Every property and developer goes through manual verification by internal team.
4. Developers can request smart home integration services even if their property is not fully integrated yet.
5. Admin must be able to approve, reject, request revision, or mark as draft.
6. Mortgage simulation must support:
